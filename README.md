# RemangaCLI

TUI клиент для [remanga.org](https://remanga.org). Сделан для личного использования - если хотите использовать, делайте это на свой риск.

Обновления не гарантированы. Код открыт - берите как основу, форкайте, переделывайте под себя, всё что угодно.

## Возможности

- Вход в аккаунт (с поддержкой 2FA)
- `select` - выбор тайтла по URL/slug, просмотр информации
- `read` - отметить главы как прочитанные (N, диапазон, +N, all)
- `like` - лайки глав
- `oc`/`cc`/`och`/`cch` - комментарии к тайтлам и главам
- `about` - информация об аккаунте
- Сохранение сессии между запусками

## Установка

```bash
pip install -r requirements.txt
python main.py
```
